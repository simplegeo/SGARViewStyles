recordAnnotationFromGeoJSONDictionary:

￼Creates a new record from the given dictionary. 

- (id<SGRecordAnnotation>) 
        recordAnnotationFromGeoJSONDictionary:(NSDictionary*)dictionary;
Parameters
geoJSONDictionary

The GeoJSON dictionary representation of the new record. ￼

Return Value

￼The SGRecordAnnotation of the GeoJSON dictionary.

Discussion

￼This method is called when the layer finds an unregistered record. The default implementation of this method creates a new instance of SGRecord and calls updateRecordWithGeoJSONDictionary:. If subclasses wish to do any manipulation to a record after it is retrieved from SimpleGeo, they are encouraged to override this method.

Properties

layerId

The layerId is the name of the layer used in SimpleGeo. A common convention used in naming a layer is a reverse URL. (e.g. "com.simplegeo.spatula").
